# Header 資源清單

本目錄存放 Header 導覽列所需的圖片資源。

---

## 必要資源

### Logo

| 檔名 | 用途 | 尺寸 | 格式 |
|------|------|------|------|
| `logo.png` | 主要 Logo（淺色背景用） | 180 x 40 px | PNG |
| `logo.svg` | 主要 Logo（向量版） | 可縮放 | SVG |
| `logo-white.png` | 白色 Logo（深色背景用） | 180 x 40 px | PNG |
| `logo-white.svg` | 白色 Logo（向量版） | 可縮放 | SVG |

### 圖示

| 檔名 | 用途 | 尺寸 | 格式 |
|------|------|------|------|
| `icon-hamburger.svg` | 漢堡選單圖示 | 24 x 24 px | SVG |
| `icon-close.svg` | 關閉選單圖示 | 24 x 24 px | SVG |
| `icon-chevron-down.svg` | 下拉箭頭 | 16 x 16 px | SVG |
| `icon-chevron-right.svg` | 子選單箭頭 | 16 x 16 px | SVG |
